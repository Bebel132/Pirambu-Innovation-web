Como usar a descrição

Este campo aceita Markdown, uma forma simples de deixar o texto mais bonito e organizado, sem precisar saber programação.

Você pode escrever normalmente e usar os botões acima para aplicar estilos ao texto, ou digitar os símbolos manualmente se preferir.

  • Negrito — destaca palavras importantes
    **texto**
  • Itálico — usado para dar ênfase
    *texto*
  • Lista — ideal para organizar informações
    - item
  • Link — adiciona um endereço clicável
    [texto](url)

Dica: você pode selecionar uma palavra ou frase antes de clicar em um botão para aplicar a formatação automaticamente.
